2. What is IE6Bar?

IE6Bar is a small widget that is meant for owners of websites that do not offer support for the Internet Explorer 6 browser anymore. As soon as you place the widget on your website, it will show up to users that still use this old browser to visit your sites. The bar notifies these users that they should upgrade to obtain support. You can try it yourself by opening the index.html from the zip archive with Internet Explorer 6.

If an user finds the bar annoying, he or she can simply click the close button. After doing this, the bar will hide itself and place a cookie to remember it should not show up anymore to this user.

2.1. Dependencies

The IE6Bar widget depends on the widely-used javascript framework called jQuery. This library provides the interface that was needed for those neat little animations and effects the warning bar has. More information about jQuery can be found at: jQuery.com.

Furthermore, it depends on the Cookie plugin for jQuery to store a cookie when the bar is hidden. This plugin was originally written by Klaus Hartl in 2006. More information about this plugin can be found at: it's project site.

Note: the dependencies are already included in the zip archive in the js folder. Though, you might want to download more recent versions when these are available. The IE6Bar widget has only been tested with jQuery v1.4.2 (minified) and jQuery Cookie Plugin v1.0.

3. Installation

The installation procedure needed to get IE6Bar to work is extremely simple. First, you need to make sure that you copy the ie6bar directory from the zip archive to the root of your site.

Second, you need to take care of the dependencies, which are jQuery and the jQuery Cookie Plugin. You can download these yourself, or just copy them from the js folder in the zip archive. Put the files somewhere in your site root or one of it's subdirectory. Just remember the path to these files, relative to the site root. If your site already uses these libraries, you can (of course) skip this step.

Third, put the following snippet of HTML in the <head>...</head> section of your site. After this, the bar should work already. 

<!--[if lte IE 6]>
<script type="text/javascript" src="js/jquery-1.4.2.min.js"></script>
<script type="text/javascript" src="js/jquery.cookie.js"></script>

<link rel="stylesheet" type="text/css" href="ie6bar/css/ie6bar.css" />
<script type="text/javascript" src="ie6bar/js/ie6bar.js"></script>
<![endif]-->

3.1. Minified files

The zip archive contains also minified versions of the javascript and css files. These files have unnecessary whitespace and comments stripped out, what will reduced the file size. This way, less bandwidth is needed and the file will be transferred faster.

The minified files are called *.min.css and *.min.js. So if you want to make use of the minified files, just use the following HTML snippet. 

<!--[if lte IE 6]>
<script type="text/javascript" src="js/jquery-1.4.2.min.js"></script>
<script type="text/javascript" src="js/jquery.cookie.min.js"></script>

<link rel="stylesheet" type="text/css" href="ie6bar/css/ie6bar.min.css" />
<script type="text/javascript" src="ie6bar/js/ie6bar.min.js"></script>
<![endif]-->

4. Configuration & Customization

IE6Bar has some configuration options, which should provide you an easy interface to customize the behaviour of the bar a little bit. Examples of customizations are text messages, supported browsers and cookie expiration time.

The configuration options can be set in the ie6bar/js/iebar.js (or ie6bar/js/iebar.min.js) file. Here you see an example of how the configuration options can be used to translate the bar: 

$(document).ready(
   function() {
      /*
       * Example configuration: Dutch translation.
       */
      var options = {
         txtTitle:
           "We merken dat je met Internet Explorer 6 surft, maar deze browser wordt door ons niet meer ondersteund...",
         
         txtSubTitle:
           "Daarom raden we je sterk aan te upgraden naar een modernere browser, zoals bijvoorbeeld Internet Explorer 8.",
         
         txtMoreInfo:
           "Meer informatie",
         
         txtExplanation:
              "Onderstaande vijf browsers worden wél ondersteund. <strong>Klik op een logo om te upgraden.</strong>",
         
         txtClose:
           "Sluiten en niet meer weergeven",
         
         
         urlIE:
           "http://www.microsoft.com/netherlands/windows/internet-explorer/",
	
         urlSafari:
           "http://www.apple.com/nl/safari/download/"
      };
      
      // Create a new ie6Bar-object and initialize the bar.
      var i6b = new ie6Bar(options);
      i6b.initialize();
   }
);

4.1. List of configuration directives

Text messages

txtTitle (string)
The large title text that appears in the bar.
Default: 
"We see you use Internet Explorer 6, but we can no longer support this browser..."
txtSubTitle (string)
The smaller subtitle text that appears just under the title text in the bar.
Default: 
"That's why we strongly recommend you to upgrade to a more modern browser, like Internet Explorer 8."
txtMoreInfo (string)
The small text that appears on the bottom of the non-expanded bar.
Default: 
"More information"
txtExplanation (string)
The small text that appears in place of txtMoreInfo when the bar expands.
Default: 
"The five browsers below are supported. <strong>Click on a logo to upgrade.</strong>"
txtClose (string)
The tooltip text of the close button.
Default: 
"Close and don't show again"

Supported browsers

txtIE (string)
The text that appears under the Internet Explorer logo on the button.
Default: 
"Internet Explorer 7+"
urlIE (string)
The URL the Internet Explorer upgrade button should point to.
Default: 
"http://www.microsoft.com/windows/internet-explorer/"
txtFirefox (string)
The text that appears under the Firefox logo on the button.
Default: 
"Mozilla Firefox 3.0+"
urlFirefox (string)
The URL the Firefox upgrade button should point to.
Default: 
"http://www.firefox.com/"
txtChrome (string)
The text that appears under the Google Chrome logo on the button.
Default: 
"Google Chrome 3.0+"
urlChrome (string)
The URL the Chrome upgrade button should point to.
Default: 
"http://chrome.google.com/"
txtSafari (string)
The text that appears under the Safari logo on the button.
Default: 
"Safari 3+"
urlSafari (string)
The URL the Safari upgrade button should point to.
Default: 
"http://www.apple.com/safari/download/"
txtOpera (string)
The text that appears under the Opera logo on the button.
Default: 
"Opera 9.5+"
urlOpera (string)
The URL the Opera upgrade button should point to.
Default: 
"http://www.opera.com/download/"

Cookie control

cookieName (string)
Name of the cookie that is set when closing the bar.
Default: 
"ie6bar"
cookieDays (int)
The number of days after which the cookie will expire. After the cookie has been expired the bar will be shown, until it is closed again.
Default: 
28
